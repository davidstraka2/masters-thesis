# Booking Client App

## Dependencies

Node.js 22.11.0 or newer, along with npm 10.9.0 or newer must be installed.

## Installation

```bash
$ npm install
```

## Environment variables

```bash
$ cp .env.template .env
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode - make sure to change the NODE_ENV env variable to NODE_ENV=production
$ npm run build
$ npm run start
```
